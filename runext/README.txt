To run the StarRCXT,

1. Preparing the command file for StarRCXT
   - Put the corresponding filenames in the template command file.
   - LEF_FILE --> public/data/libraries/lef/*.lef
   - TOP_DEF_FILE --> aes_cipher_top/aes_cipher_top.def
   - TCAD_GRD_FILE --> nxtgrd/*.nxtgrd
   - MAPPING_FILE --> layermap/*.map
    
2. Running StarRCXT
   % StarXtract star_wst.cmd
